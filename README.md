# Animal-Emotion-Recognition-Multimodal-Deep-Learning-System

Developed an Explainable Multimodal System using Wav2Vec2 for audio, Vision Transformer (OpenCLIP) for images & biometric sensors.
The system employs LSTM & dense neural networks for multimodal fusion & uses GPT-4 to generate natural language explanations. 
Built with Flask backend & web frontend for real-time emotion classification and visualization.
